# Countries Data Scraper 🌍

A Python web scraping project designed to extract a comprehensive list of countries along with their capitals, population, and total area from the "Scrape This Site" sandbox.

## Features
- **Single-page Extraction:** Quickly parses the layout to gather all country profiles in one execution.
- **Null Safety:** Features protective conditions to handle missing tags gracefully.
- **Clean Structure:** Outputs to a well-formatted `Country.csv` spreadsheet.

## Data Fields Extracted
- Country Name
- Capital City
- Population
- Total Area (sq km)

## Tech Stack
- Python 3
- Requests
- BeautifulSoup4
- Pandas
